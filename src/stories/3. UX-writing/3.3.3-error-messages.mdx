# 3.3.3 오류 메시지 (Error Messages)

- 오류가 발생한 원인을 시스템 용어로 어렵게 설명하지 않으며, 사용자의 잘못으로 돌리는 표현(예: "잘못 입력하셨습니다")을 지양합니다.
- 무엇이 문제인지, 그리고 **어떻게 해결할 수 있는지(Actionable)**를 구체적으로 안내합니다. 
- 예: "비밀번호가 일치하지 않습니다. 다시 한 번 확인해 주세요." 또는 "영문자와 숫자를 조합하여 8자 이상 입력해 주세요."
