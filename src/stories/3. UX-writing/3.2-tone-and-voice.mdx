# 3.2 톤 & 보이스

사용자에게 전달하고자 하는 우리 서비스의 성격과 감정적인 어조를 정의합니다.

## 3.2.1 Voice (목소리)
우리의 목소리는 변하지 않는 브랜드의 근본적인 성격과 가치관입니다.
- **전문적이고 명확한**: 사용자가 신뢰할 수 있도록 전문적이고 정확한 정보를 전달합니다.
- **친근하고 배려하는**: 너무 딱딱하거나 차갑지 않게, 사용자를 존중하고 공감하는 태도를 유지합니다.
- **투명하고 솔직한**: 숨기는 것 없이 명확하게 상황을 설명하며 투명한 서비스를 제공합니다.

## 3.2.2 Tone (어조)
어조는 상황과 문맥, 사용자의 현재 감정 상태에 따라 유연하게 변합니다. 

- **긍정적인 상황 (예: 작업 성공, 가입 완료, 축하)** 
  - 밝고 격려하는 어조를 사용합니다. 사용자의 성취에 공감합니다.
- **부정적인/심각한 상황 (예: 결제 오류, 시스템 장애, 데이터 삭제)** 
  - 명확하고 담백하며, 해결책을 우선시하는 어조를 사용합니다. 감정적인 표현은 배제하고 사실 전달에 집중합니다.
- **중립적인 안내 (예: 설정 변경, 기본 툴팁)** 
  - 객관적이고 간결한 어조를 사용하여 사용자가 정보를 정확히 인식하도록 돕습니다.
