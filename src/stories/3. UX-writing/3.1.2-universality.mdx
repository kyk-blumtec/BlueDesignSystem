# 간결성

### 필요한 정보만 전달
<ul className="guide-list">
  <li>핵심 정보만 남기고, 사용자의 이해에 불필요한 단어·군더더기를 제거합니다.</li>
  <li>짧게 표현하기 위해 꼭 필요한 정보를 생략하지 않습니다.</li>
  <li>짧지만 의미가 명확한 문장으로 한 화면에서 읽어야 하는 인지 부담을 줄여야 합니다.</li>
</ul>

<ul className="guide-list">

  <li>한글과 한자어를 함께 쓸 때 의미가 겹치지 않는지 확인합니다.</li>
</ul>

### 약어의 사용
<ul className="guide-list">
  <li>약어만 쓰고 풀이나 설명이 없으면 무엇을 의미하는지 사용자가 파악하기 어렵습니다. 외국어 단어의 약어를 쓸 때는 약어 풀이도 함께 씁니다.</li>
  <li>제목에는 약어 풀이를 쓰지 말고 약어만 쓰며, 한 화면을 기준으로 본문에 처음 나올 때 약어와 함께 약어 풀이를 씁니다. 한 화면을 기준으로 두 번째 나올 때 부터는 약어만 씁니다.</li>
</ul>
