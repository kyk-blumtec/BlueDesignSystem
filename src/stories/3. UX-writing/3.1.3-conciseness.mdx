# 일관성

### 일관된 UI 텍스트 사용
<ul className="guide-list">

### 한글 맞춤법의 띄어쓰기 규정 준수
<ul className="guide-list">
  <li>서비스의 버튼이나 메뉴, 탭 이름 등은 메시지를 표시할 공간이 부족할 경우 붙여 쓸 수 있습니다. 전문용어도 단어별로 띄어 쓰는 것이 원칙이지만 붙여 쓸 수 있습니다.</li>
  <li>단, 서비스 전체에서 용어 별로 띄어 쓰기 일관성을 유지하는 것이 중요합니다.</li>
</ul>

#### ① 띄어쓰기 기본 원칙
<p style={{ marginTop: '8px', marginBottom: '0px', color: '#334155' }}>
  한글 맞춤법은 “문장의 각 단어는 띄어 씀을 원칙으로 한다.” 라고 규정되어 있습니다. 
</p>

      <td>
        <div><span className="text-good">[O]</span> 작은 창 사전</div>

        이 때 한 단어로 취급되는지는 표준국어대사전 등재 여부로 판단합니다.
      </td>

        <div><span className="text-good">[O]</span> 찾아본 의약품</div>

#### ② 조사와 접사는 앞 말에 붙여 씁니다.
<p style={{ marginTop: '8px', marginBottom: '0px', color: '#334155', lineHeight: '1.6' }}>
  띄어쓰기 원칙에 따르면 조사도 단어이므로 띄어 써야 하지만, 다른 단어에 비해 독립성이 떨어지기 때문에 예외적으로 앞 말에 붙여 쓰도록 합니다.<br />
  접사는 단독으로 쓰이지 않고 항상 다른 단어에 붙어 새로운 단어를 구성합니다.
</p>

#### ④ 의존명사는 앞 말과 띄어 씁니다.
<p style={{ marginTop: '8px', marginBottom: '16px', color: '#334155' }}>
  의존명사는 앞 말과 띄어 씁니다. UI 텍스트에서 자주 나오는 의존명사에는 ‘님’, ‘중’, ‘시’ 등이 있습니다.
</p>
