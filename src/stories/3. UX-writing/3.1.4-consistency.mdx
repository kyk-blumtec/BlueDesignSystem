<ul className="guide-list">

</ul>

### 사람이 하는 동작은 능동형 사용
<ul className="guide-list">
  <li>사람이 하는 동작은 능동형으로 써야 사용자 관점에서 더 자연스럽습니다. 영어를 번역할 때는 원문에 피동형이 쓰였더라도 능동형으로 번역합니다.</li>
</ul>
